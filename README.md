# ⚔️ TRANSCRIPT IMPERIUM ⚔️

**For the Emperor! Extract YouTube transcripts with the fury of the Adeptus Mechanicus!**

## 🎯 Features

- **Single Video Mode**: Extract transcript from one YouTube video
- **Full Channel Mode**: Extract all transcripts from an entire YouTube channel
- **Warhammer-themed UI**: Dark, gothic interface inspired by the Imperium
- **Real-time Progress**: Live progress bar during extraction
- **Structured Output**: Clean .txt format with video metadata

## 🚀 Usage

1. Open the web interface: https://kioka8877-ux.github.io/transcript-imperium/
2. Paste your YouTube URL (video or channel)
3. Select mode (Single Video or Full Channel)
4. Click "INITIATE EXTRACTION"
5. Wait for the progress bar to complete
6. Download your transcript file

## 🛠️ Technical Stack

- **Frontend**: HTML + CSS + JavaScript (GitHub Pages)
- **Backend**: Python + GitHub Actions
- **Library**: youtube-transcript-api (no API key required)
- **Storage**: GitHub Releases
